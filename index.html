How does my browser render websites?

Websites are a combination of HTML, CSS, and Javascript

  • HTML provides the basic structure and architecture of a webpage
  • CSS gives colors, formatting, and layout to websites
  • Javascript gives websites functionality and makes them interactive

What is Harriscript?

Harriscript is a CSS Preprocessor written in Javascript

CSS compared to Harriscript

$color blue

$size 12px

 color red

 color: red;

 font-size: 12px

 color: red;

 font-size: 12px;
